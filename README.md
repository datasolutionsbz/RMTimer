# RMTimer
RM Timer - Desktop Time Tracker for Redmine

A minimalist desktop application that makes tracking time in Redmine fast and effortless. No more switching browser tabs or navigating complex web interfaces—track time with a single click.

✨ Features
One-Click Time Tracking - Start and stop timers with a single click per issue
System Tray Integration - Runs minimized, accessible from your system tray
Direct Redmine Sync - Logs time directly to your Redmine instance via API
Clean Interface - Simple list of your assigned issues without distractions
Multi-Platform - Available for Windows, macOS, and Linux
Lightweight - Minimal resource usage, runs in the background

Get Your Redmine API Key:
Log into your Redmine instance
Go to My Account → API access key
Click "Show" and copy your key

Configure RM Timer:
Launch the application
Go to Settings
Enter your Redmine URL (e.g., https://redmine.yourcompany.com)
Paste your API key
Click Test Connection

Start Tracking:
Your assigned issues will appear in the main window
Click the ▶️ Start button next to any issue
Click ⏹️ Stop when done - time is automatically logged!
